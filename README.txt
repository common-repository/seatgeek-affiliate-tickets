== Description ==
Show events and affiliate tickets from Seatgeek website using a simple shortcode, you can also add your affiliate link so you can earn money when anyone buys a ticket from your website.

You can choose any performer to get his events tickets, it's also working with Ajax load more for pagination.

https://www.youtube.com/watch?v=pv_TNphK70U

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `seatgeek-affiliate-tickets.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
